Triple Match

The goal of the game is to keep the board from filling. It's inspired by Triple Town from Spryfox, so you won't be lost if you played this one before.

Each turn you will be presented a new item to place in the "Current" box on the left. Click on the cell where you want to place this item.

The "Stash" under the "Current" box allows you to store an item for later use. Click on the stash to move the current item into the stash (and recover the stashed item).

Specials

The "star" is a joker. It matches any other block or diamond. If two or more matches are possible, the "smaller" match will be done.

The "key" is an eraser. It removes anything.

Triple Match

Le but du jeu est d'empêcher le jeu de se remplir. Le jeu est inspiré de Triple Town du studio Spryfox, vous ne serez donc pas perdu si vous y avez déjà joué.

A chaque tour, le jeu vous présente un nouvel élément dans la zone "Current" à gauche. Cliquez dans l'espace de jeu sur la cellule où vous voulez placer cet élément.

La zone "Stash" sous la zone "Current" vous permet de conserver un élément que vous ne voulez pas utiliser immédiatement. Cliquez sur le "Stash" pour y stocker l'élément courant (et récupérer l'élément stocké).

Éléments spéciaux

"L'étoile" est un joker. Elle peut se bombiner avec n'importe quel bloc ou n'importe quel diamant. Si deux combinaisons (ou plus) sont possibles, la "plus petite" sera choisie.

La "clé" est une gomme. Elle peut enlever n'importe quel élément.
